#　横スクロールシューティング

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲーム概要
主人公をキーボード操作で何かを発射し敵を倒すゲーム

## ゲーム実装
### 共通基本機能
* 背景画像と主人公キャラクターの描画　msou_kokatonがベースになっている

### 担当追加機能
* ボスについての機能（担当：市川）：条件を満たすと通常の敵より強く設定された敵（ボス）が登場する機能 弾を２０回ヒットさせると倒す  通常の敵より大きめの爆弾を高頻度で自機に向かって飛ばす
